## ğŸ“Š PRD Levels Overview

The template supports three levels of detail:

### Level 1: Basic PRD

- **Time:** 1-2 hours
- **Use case:** MVPs, prototypes, experiments
- **Sections:** Overview + Goals + Features + Timeline

### Level 2: Epic + User Stories

- **Time:** Iterative, continuous evolution
- **Use case:** High uncertainty projects, pure Agile teams
- **Format:** Epic + User Stories + Product Backlog

### Level 3: Complete PRD

- **Time:** 1-2 days
- **Use case:** Complex products, regulated industries, established projects
- **Sections:** All sections in the template

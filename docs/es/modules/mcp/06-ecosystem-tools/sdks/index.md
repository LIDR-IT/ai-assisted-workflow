# SDKs Oficiales de MCP

Kits de desarrollo oficiales para crear servidores MCP.

## SDKs Disponibles

### [TypeScript SDK](typescript-sdk.md)
SDK oficial para TypeScript y JavaScript
- Type-safe APIs
- Soporte completo de transporte
- Helpers para tools, resources, prompts
- Validación automática

### [Python SDK](python-sdk.md)
SDK oficial para Python
- API pythonic
- Async/await support
- Type hints completos
- Decorators para tools

## Instalación Rápida

### TypeScript

```bash
npm install @modelcontextprotocol/sdk
```

### Python

```bash
pip install mcp
```

---

## Eligiendo un SDK

### Usa TypeScript si:
- Trabajas con Node.js o Deno
- Necesitas integración con ecosistema npm
- Prefieres type safety estricto
- Desarrollas para web

### Usa Python si:
- Trabajas con data science/ML
- Prefieres sintaxis simple
- Integras con librerías Python
- Inicias en desarrollo de servidores
